jquery.jui.checkboxes
=====================

Creates accessible custom checkboxes

Usage:

HTML:
&lt;span class="jui-checkbox" tabindex="0"&gt;&lt;/span&gt;

Initialize the selector. JavaScript:
$("#myCheckboxes").juicheckboxes();
